Projects

Realtime ASCII Art Camera
This fun project captures live video from a camera and converts it to ASCII art on the fly. It merges creativity with programming, creating a unique visual experience in terminal or GUI. A great way to explore image processing and character rendering techniques.
Air Quality Monitoring System
Built a sensor-based system that measures and monitors air pollution in real-time. The system includes a live data visualization dashboard and email alert notifications to keep users informed. It combines embedded systems, Java, and cloud integration.
Email Reply Generator
This project automatically generates personalized email replies by analyzing incoming messages. It uses natural language processing to speed up email responses and improve productivity. A handy tool for professionals drowning in emails.
Ping Pong Game with Stats
A simple browser-based Ping Pong game featuring real-time user stats, including highest score tracking. Developed to experiment with game mechanics, UI, and JavaScript performance. Fun and challenging way to showcase interactive programming skills.
